# Placeholder Media Slots

Este directorio define la estructura estable de assets para reemplazo visual por modelo:

- `public/products/generated/<slug>/hero.jpg`
- `public/products/generated/<slug>/detail-01.jpg`
- `public/products/generated/<slug>/detail-02.jpg`

Cuando lleguen fotos oficiales, se sustituyen estos archivos manteniendo los mismos nombres.
